### Prerequisiste
You should have a <b>linux</b> system with a <b>python</b> version greater than 3. 

### Installation
Run `install.sh` as sudo
```
$ git clone https://github.com/Yashvendra/Ogre.git
$ cd Ogre
$ chmod +x install.sh
$ sudo ./install.sh
```
### Usage
Just type `ogre` in the shell.
```
$ ogre
```
